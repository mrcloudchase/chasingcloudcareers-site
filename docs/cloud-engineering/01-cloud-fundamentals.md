# Cloud Fundamentals

## Cloud Computing Core Concepts
- **What you Need to Know**
  - **Cloud Service Models and Deployment Types**
    - Infrastructure as a Service (IaaS) - Virtual machines, storage, networking
    - Platform as a Service (PaaS) - Application platforms and development environments
    - Software as a Service (SaaS) - Complete applications delivered over the internet
    - Public, private, hybrid, and multi-cloud deployment strategies
    - **Resources:**
      - [Cloud Computing Fundamentals - NIST](https://csrc.nist.gov/publications/detail/sp/800-145/final) - Official cloud computing definition and characteristics
      - [Cloud Service Models Explained - AWS](https://aws.amazon.com/types-of-cloud-computing/) - IaaS, PaaS, SaaS comparison and use cases
      - [Cloud Deployment Models - Microsoft](https://docs.microsoft.com/en-us/learn/modules/fundamental-azure-concepts/) - Public, private, hybrid cloud strategies

  - **Cloud Economics and Pricing Models**
    - Pay-as-you-go pricing and consumption-based billing
    - Reserved instances and committed use discounts
    - Spot instances and preemptible VMs for cost optimization
    - Total Cost of Ownership (TCO) analysis and cost management
    - **Resources:**
      - [Cloud Economics - AWS](https://aws.amazon.com/economics/) - Cloud cost optimization strategies and TCO analysis
      - [Azure Cost Management](https://docs.microsoft.com/en-us/azure/cost-management-billing/) - Cost optimization and billing management
      - [Google Cloud Pricing](https://cloud.google.com/pricing) - GCP pricing models and cost optimization tools
